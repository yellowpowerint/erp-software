# Getting Started

---

# Getting Started

**Mining ERP** is a company system on a computer.

You use it to:
- Track inventory (warehouse items)
- Record operations and production
- Manage vehicles and equipment
- Request and approve purchases and payments
- Manage employees and safety
- View reports

---

# Login (Step-by-Step)

1. Open browser (Chrome / Firefox / Edge)
2. Enter website address from IT
3. Enter your email + password
4. Click **Login**

**Important:** Keep your password secret.

---

# First Time Login

If the system asks you to change password:

1. Type old password
2. Type new password
3. Confirm new password
4. Click **Change Password**

# Approvals

---

# Module 1: Approvals & Workflows

Used for requests that need permission.

Common types:
- Invoice approvals
- Purchase requests
- IT requests
- Payment requests

---

# Approvals: Check 5 Things (Rule)

Before you approve, check:

1. Is it real? (not a mistake)
2. Is the amount correct?
3. Is there proof? (invoice/quote/receipt/delivery note)
4. Do we need it now?
5. Is it within budget?

---

# Approvals: How to Find Your Tasks

1. Go to **Approvals**
2. Click **My Pending Approvals**
3. Open a request
4. Approve or Reject

**Reject:** Always write a reason.

# Inventory & Assets

---

# Module 2: Inventory & Assets

**Inventory:** items stored in warehouses

**Assets:** big items (trucks, excavators, generators)

---

# Inventory: Stock In (Items Arrive)

1. Inventory → Manage Stock
2. Stock In
3. Select item + quantity + warehouse
4. Add reason
5. Upload proof (delivery note/receipt)
6. Save

---

# Inventory: Stock Out (Items Issued)

1. Inventory → Manage Stock
2. Stock Out
3. Select item + quantity + reason
4. Add who took it
5. Save

---

# Inventory: Stock Alerts + Stock Taking

**Stock Alerts**
- Inventory → Stock Alerts
- See low items
- Inform Procurement / create request

**Stock Taking (Counting)**
- Count shelf items
- Compare with system
- Report differences

# Module 5: Finance & Procurement

Finance manages money.

Procurement manages buying.

Key areas:
- Budgets
- Expenses
- Payments
- Procurement steps

---

# Procurement Steps (Simple)

1. Requisition (request to buy)
2. RFQ (ask vendors prices)
3. Purchase Order (order)
4. Goods Receipt (receive)
5. Invoice (vendor bill)
6. Payment

---

# Finance: Match the 3 Papers (Rule)

Before paying, match:

1. Purchase Order
2. Goods Receipt / Delivery Note
3. Invoice

If they do not match: stop and ask your manager.

# Common Tips

- If you can’t see a page: role access may be missing
- Always upload proof (receipts, invoices, delivery notes)
- Write clear descriptions
- Check notifications daily
- Log out when done

---

# Troubleshooting

**Can’t log in**
- Check spelling
- Caps Lock
- Use “Forgot Password” (if available)
- Contact IT

**Page not loading**
- Refresh
- Check internet
- Try another browser
