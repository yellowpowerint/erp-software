## Getting Started

### What is Mining ERP?

Mining ERP is a work system on a computer.
It helps the company do work in one place.

You can use it to:
- Track inventory (warehouse items)
- Record operations and production
- Manage vehicles and equipment
- Request and approve purchases and payments
- Manage employees and safety
- View reports

### How to Log In

1. Open your web browser (Chrome, Firefox, or Edge)
2. Type the website address given by IT
3. Enter your email and password
4. Click **Login**

**Important:** Keep your password secret.

### First Time Login

If the system asks you to change your password:
1. Type your old password
2. Type your new password
3. Type the new password again
4. Click **Change Password**

### Understanding Your Role (The 12 Roles)

Your **role** controls what you can see and do in the system.

These are the 12 roles:

1. **SUPER_ADMIN** - Can do everything in the system
2. **CEO** - Executive oversight and approvals
3. **CFO** - Finance oversight and approvals
4. **DEPARTMENT_HEAD** - Manages a department and approvals
5. **ACCOUNTANT** - Records expenses, payments, and finance work
6. **PROCUREMENT_OFFICER** - Buying, vendors, RFQs, purchase orders
7. **OPERATIONS_MANAGER** - Operations, projects, production, shifts
8. **IT_MANAGER** - System setup, users, permissions, support
9. **HR_MANAGER** - Employees, attendance, recruitment, leave
10. **SAFETY_OFFICER** - Incidents, inspections, training, compliance
11. **WAREHOUSE_MANAGER** - Inventory, stock in/out, stock alerts
12. **EMPLOYEE** - Basic access (daily tasks and requests)

If you cannot see a page, it usually means your role does not have access.

## Module 1: Approvals & Workflows

Approvals are permission steps.

### Common approval types

- Invoice approvals
- Purchase requests
- IT requests
- Payment requests

### How to see your approvals

1. Go to **Approvals**
2. Click **My Pending Approvals**

### Simple Approval Rule (Check 5 Things)

Before you approve, check:
1. **Is it real?** (Not a mistake)
2. **Is the amount correct?**
3. **Is there proof?** (invoice, quote, receipt, delivery note)
4. **Do we need it now?**
5. **Is it within budget?**

### Approve or Reject

- To approve: open the request, click **Approve**, add comment (if needed), confirm.
- To reject: click **Reject**, write a reason, confirm.

## Module 2: Inventory & Assets

Inventory is items stored in a warehouse.
Assets are big items like trucks, excavators, generators.

### Stock In (items arrive)

1. Go to **Inventory**
2. Choose **Manage Stock**
3. Choose **Stock In**
4. Select item, quantity, warehouse, reason
5. Upload receipt/delivery note (if you have it)
6. Save

### Stock Out (items issued)

1. Go to **Inventory** → **Manage Stock**
2. Choose **Stock Out**
3. Select item, quantity, reason, who took it
4. Save

### Stock Alerts (low stock)

1. Go to **Inventory** → **Stock Alerts**
2. See items running low
3. Tell Procurement or create a request

### Stock Taking (Counting Stock)

Stock taking means counting the shelf items and comparing with the system.

1. Open the item list for the warehouse
2. Count items
3. Compare with system quantity
4. If different, report to your supervisor

### Assets (heavy equipment)

- View assets in **Assets**
- Each asset has status: Active, Maintenance, Retired
- You can schedule maintenance when needed

## Module 5: Finance & Procurement

This module manages money and buying.

### Budgets

- Create budgets per department
- Track how much is spent and remaining

### Expenses

- Record each expense
- Attach receipt/invoice

### Payments

- Create a payment request
- It goes for approval
- After approval, it can be marked as paid

### Procurement steps (simple)

1. Requisition (request to buy)
2. RFQ (ask vendors for prices)
3. Purchase Order (order items)
4. Goods Receipt (receive items)
5. Invoice (vendor bill)
6. Payment

### Simple Rule: Match the 3 Papers

Before paying, match:
1. Purchase Order
2. Goods Receipt / Delivery Note
3. Invoice

If they do not match, stop and ask your manager.

## Appendix: Detailed Step-by-Step Guides

Use this appendix when you want the **full step-by-step** for each module.

### A1. Approvals & Workflows (Detailed)

**How to see pending approvals**
1. Click **Approvals**
2. Click **My Pending Approvals**
3. Open any request to see details

**How to approve**
1. Open the request
2. Read the details and check proof
3. Click **Approve**
4. Add a comment (optional)
5. Click **Confirm**

**How to reject**
1. Open the request
2. Click **Reject**
3. Write a reason (required)
4. Click **Confirm**

**Common statuses (simple meaning)**
- **Pending** = waiting for approval
- **Approved** = accepted
- **Rejected** = not accepted
- **Paid** = money paid (for payments/invoices)

### A2. Inventory & Assets (Detailed)

**How to view inventory**
1. Click **Inventory**
2. Use search to find an item
3. Use filters (category/warehouse) if needed

**What item fields mean**
- **Item Code** = item ID (example: ITEM-001)
- **Current Quantity** = how many we have now
- **Reorder Level** = when it becomes low stock

**Stock In**
1. Inventory → Manage Stock → Stock In
2. Choose item and warehouse
3. Enter quantity
4. Add reason (example: Supplier delivery)
5. Upload delivery note/receipt if available
6. Save

**Stock Out**
1. Inventory → Manage Stock → Stock Out
2. Choose item and warehouse
3. Enter quantity
4. Add reason (example: Issued to Site A)
5. Add who collected the item
6. Save

**Stock Alerts**
1. Inventory → Stock Alerts
2. See items below reorder level
3. Tell Procurement or create a requisition

### A3. Fleet Management (Detailed)

**Vehicle assignments (give vehicle to driver)**
1. Fleet → Assignments
2. Click **New Assignment**
3. Select vehicle and driver
4. Enter start date and purpose
5. Save

**Fuel record**
1. Fleet → Fuel → Record Fuel
2. Select vehicle
3. Enter liters and cost
4. Upload receipt photo if available
5. Save

**Breakdowns**
1. Fleet → Breakdowns → Report Breakdown
2. Select vehicle, location, and problem
3. Choose severity
4. Upload photos
5. Submit

### A4. Operations (Detailed)

**Production logs**
1. Operations → Production Logs
2. Click **New Log**
3. Enter date, shift, site, material, quantity
4. Save

**Equipment usage**
1. Operations → Equipment Usage
2. Record start/end time and hours used
3. Save

**Shift planning**
1. Operations → Shift Planning
2. Create schedule for dates
3. Assign employees
4. Publish

### A5. Finance & Procurement (Detailed)

**Budgets**
1. Finance → Budgets
2. Click **New Budget**
3. Enter department and amount
4. Save

**Expenses**
1. Finance → Expenses → New Expense
2. Enter description, amount, budget
3. Upload receipt
4. Save

**Procurement flow**
1. Procurement → Requisitions (request items)
2. Procurement → RFQs (ask vendors prices)
3. Procurement → Purchase Orders (order)
4. Procurement → Goods Receipts (receive)
5. Finance/Procurement → Invoices (record bill)
6. Finance → Payments (pay after approval)

### A6. HR (Detailed)

**Attendance**
1. HR → Attendance
2. Choose date
3. Mark present/absent/leave
4. Save

**Leave requests**
1. HR → Leave Requests
2. New request
3. Choose dates and reason
4. Submit

### A7. Safety & Compliance (Detailed)

**Incident report**
1. Safety → Incident Reports → Report Incident
2. Enter date, location, description
3. Upload photos
4. Submit

**Near miss**
Report it the same way as an incident.

**Compliance documents**
1. Safety → Compliance Docs → Upload
2. Enter expiry date
3. Save

### A8. AI Tools (Detailed)

**Knowledge Engine (Ask a question)**
1. AI → Knowledge Engine
2. Type a question
3. Click Ask
4. Read the answer and follow company rules

### A9. Reports (Detailed)

1. Reports → choose report type
2. Select date range
3. Generate
4. Download PDF/Excel if needed

### A10. Documents (Detailed)

**Upload**
1. Documents → My Uploads
2. Upload
3. Add name, category, tags
4. Save

**Sharing**
1. Open a document
2. Share
3. Choose who can view/edit
4. Confirm

### A11. Projects (Detailed)

1. Projects → New Project
2. Enter name, dates, manager
3. Save
4. Add tasks and milestones

### A12. Settings (Detailed)

Only for admins/managers.

1. Settings → Users & Roles
2. Create or edit users
3. Save permissions

## Troubleshooting

- **Cannot log in**
  - Check email spelling
  - Check Caps Lock
  - Use "Forgot Password" if available
  - Contact IT

- **Page not loading**
  - Refresh
  - Check internet
  - Try another browser
  - Contact IT

- **Upload failed**
  - Try smaller file
  - Check internet
  - Contact IT
